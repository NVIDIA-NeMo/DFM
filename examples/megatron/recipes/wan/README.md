# Wan Recipes

Recipes for training and inferencing Wan models using Megatron-Core.

## Files

- `pretrain_wan.py`: Main pre-training script.
- `inference_wan.py`: Inference script.
- `prepare_energon_dataset_wan.py`: Dataset preparation utility.
